# IDR | Institute of Digital Risk

An industry-led institute for training and deploying digital risk practitioners.

## 🚀 Live Demo
[View Live Site](https://shradddha30.github.io/idr-institute/)

## 🎨 Design Choices
- **Logo**: A geometric cube representing structure, resilience, and multi-faceted digital risk.
- **Color Palette**: 
  - **Orange (#FF6B00)**: Innovation and energy.
  - **Black & White**: Professional authority and minimalist tech aesthetic.
- **Typography**: Clean Sans-serif for high legibility across all devices.

## 🛠️ Built With
- **HTML5**: Semantic structure.
- **CSS3**: Custom layout using Flexbox and CSS Grid (No Frameworks).
- **Vanilla JavaScript**: For smooth scrolling and form interactions.

## 📁 Sections
- **Hero**: Core mission statement and CTA.
- **About**: IDR’s bridge between academia and industry.
- **Services**: Academy, Innovation, and Advisory pillars.
- **Contact**: Interest registration form with LocalStorage integration.
